# Prompt - Build DocPulse

Use this prompt for future implementation work.

I want to build a full SaaS Doctor Appointment System called **DocPulse** using:

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT + HTTP-only Cookies

## Important Rule

Do not implement the app yet. Use the documentation files as the source of truth first.

## Product Summary

DocPulse helps patients book doctor appointments and helps clinics manage doctors, schedules, patients, reminders, and reports.

## Version 1 Scope

- Landing page
- Authentication
- Patient dashboard
- Doctor dashboard
- Clinic admin dashboard
- Doctor listing and profile
- Appointment booking
- Appointment management
- Doctor availability
- Basic email notification

## Design Direction

- Modern healthcare SaaS
- White background
- Blue, sky, and teal gradient
- Rounded cards
- Soft shadows
- Glassmorphism
- Smooth animations
- Mobile-first responsive layout

## Implementation Rules

- Use Next.js for the frontend
- Use Express.js for the backend
- Use MongoDB for persistence
- Use JWT stored in HTTP-only cookies
- Keep TypeScript clean
- Build a professional SaaS experience
