# AI Context - DocPulse

## Project Identity

Project name: **DocPulse**

Project type: **SaaS Doctor Appointment System**

Main goal: Build a secure, modern healthcare platform where patients can book doctor appointments and clinics can manage schedules, appointments, doctors, patients, reminders, and reports.

## Final Stack

- Frontend: Next.js + React + TypeScript + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB + Mongoose
- Authentication: JWT + HTTP-only Cookies
- UI: shadcn/ui + Radix UI + Lucide React
- Animation: Framer Motion
- Email/Reminder: Nodemailer
- File Upload: Cloudinary

## Product Roles

- Patient
